How to use this file

  1. Tasks are grouped by epic (large feature or theme)
  2. Each task has a unique ID — never reuse IDs, even if a task is deleted
  3. When starting work, update Status and Owner
  4. When done, move the task to the Completed section at the bottom
  5. Keep this file in sync with PRs — reference task IDs in commit messages and PR descriptions
